Wallpaper

The Living Dusk follows your clock

Both monitors render the studio's Hill-Country dusk as a live scene — drifting valley mist, rising embers, god-rays, shooting stars — drawn natively at each monitor's exact resolution. It isn't a video; it's the brand painted in real time, and it changes with the actual hour:

DawnGolden afternoonSignature duskDeep night

After dark the embers become fireflies and a faint Milky Way appears. Every login plays a 2.6-second sunrise intro.

Facts worth knowing

  • Cost: ~2–3% CPU total, and it auto-pauses whenever a fullscreen game runs — zero cost while you play.
  • Master file: Pictures\Liberty Hill Studios\lhs-dusk.html — Lively runs a synced copy.
  • Pin a mood forever: add ?mood=dusk (or night / day) to the wallpaper URL in Lively.
  • Lively tray icon (bottom-right): pause, change, or close the wallpaper anytime.

Zones

Shift-drag: windows snap like they mean it

FancyZones gives each monitor named parking spots. The ultrawide runs LHS Ultrawide — a 40% center stage with two 30% wings. The 4K runs LHS Quad. Watch the motion:

Shift held
Drag → hold Shift → gold zones appear → drop on one (or on the border between two to span both).

The four motions

  • Snap: drag a window, hold Shift, aim the cursor at a zone, release.
  • Span: same, but aim at the border between zones — both light up, window fills both. That's your 70% mega-editor.
  • Keyboard: Win+←/→ walks the focused window zone-to-zone — and across monitors at the edge.
  • Memory: apps reopen in their last zone automatically. Drag without Shift to pop a window out at its original size.

Change a monitor's layout anytime: Win+Shift+` opens the editor on whichever screen your cursor is on — the LHS layouts live under "Custom".

Try it — grab this browser window, hold Shift, drop it in the left wing. Then Win+→ twice.
Search

Everything, everywhere, instantly

Everything indexes every file on all three drives by reading NTFS directly — results appear faster than you can finish typing. It powers three surfaces:

  • The launcher (Alt+Space) — file results are Everything results.
  • The taskbar — Everything Toolbar replaces the old search box (if its 2-click setup assistant is still waiting, finish it from its tray icon).
  • The full app — launch Everything itself for power filters: *.blend dm:today, size:>1gb, parent:"C:\Users\derek\Documents\GitHub".
Try it — Alt+Space and type lhs-dusk.html. Instant, from a cold start.
Shell

The taskbar lives inside the scene

TranslucentTB keeps the taskbar fully transparent, so the god-rays and embers drift behind your icons all day. The wallpaper bakes a quiet ink strip along the bottom so icons stay readable no matter where the sun is. Explorer got sharper too:

  • Win+E opens straight to This PC — all drives front and center, with Liberty Hill (C:) leading.
  • Quick Access pins: GitHub, the liberty-hill-studios repo, and the studio art folder — all wearing the Lone Star.
  • This PC sits on the desktop as a permanent anchor.
  • Start menu and windows stay translucent dark — gold appears only as accents (toggles, selection, the terminal cursor). Never as a background. That's law.
Sound

Studio chimes — strictly opt-in

The kit ships two short chimes (a warm "finished" note and a low "needs you" bell), but they're off by default — system sounds are personal, and they never replace yours unless you ask. Opt in with .\install.ps1 -Chimes; the WAVs live in sounds\ for auditioning first.

Finishing touches

Optional finishing touches

  • Gold pointer — Settings → Accessibility → Mouse pointer → Custom style → Choose another color → #E8A13A. Gold cursor everywhere, natively crisp at any size.

  • Account avatar — Settings → Accounts → Your info → Choose a file → icons\avatar-448.png. Puts the glowing Lone Star in Start, login, and every UAC prompt. (Requires activated Windows.)

  • Everything Toolbar pin — right-click its taskbar icon → Pin to taskbar. Windows only allows this by hand.
